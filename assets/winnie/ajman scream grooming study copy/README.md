# Ajman screaming and grooming study

This self-contained extension contains two eight-frame transparent animation loops for Ajman:

- `screaming`: alert, inhale, mouth opening, full yowl, and recovery.
- `grooming`: lift the striped left paw, lick it, rub the face and tipped ear, then settle.

## Main files

- `final/ajman-scream-groom-atlas.webp`: two-row transparent action atlas, 192×208 cells.
- `frames/`: individual transparent PNG frames.
- `previews/`: animated GIF previews.
- `qa/contact-sheet.png`: both rows on an exact magenta review field.
- `manifest.json`: action names, row assignments, timing, and identity rules.

The installed Ajman pet was read only as a reference and was not modified.
