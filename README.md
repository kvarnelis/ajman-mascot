# Ajman

Ajman is a small macOS desktop companion starring two real cats: Ajman, a tuxedo, and Winnie, a tabby. They appear as pixel-art pets on your desktop and react as Claude Code and Codex work, wait, finish, or need attention.

## Install

Requires macOS 13 or later. Open the Ajman DMG, drag **Ajman.app** onto **Applications**, then open Ajman from Applications.

On first launch, Ajman asks whether he should launch at startup. You can change that choice later from **Launch at Login** in the menu-bar menu.

## Use

Ajman lives in the menu bar under the cat icon. From there you can:

- choose animations from **Actions**, including the agent states, **Sleep**, and **Scratch**;
- set each cat's **Temperament**, from Catatonic to Insane;
- show or hide cats under **Pets**, change **Overall Size**, or **Reset Position**;
- toggle **Launch at Login** and **Show agent notifications**; and
- **Connect to Claude Code**.

Control-click or right-click a cat to step just that cat through its actions without changing the other one.

**Connect to Claude Code** adds Ajman's event hooks to `~/.claude/settings.json`, letting the cats hear what Claude Code is doing. Untick it to remove Ajman's hooks again.

## A note about the cats

The artwork depicts the owner's real cats. Please enjoy it as part of Ajman, but do not redistribute or regenerate the cat art.
